# Overview
Android speech-to-text transcriber

Created by Ravi Patel and Dhakshna Morthy

# Main Features
  - Google voice typing
  - Export option

# QoL
  - Optional, editable prefix to transcribed text
  - Clearing transcript
  - Persistent state with automatic line separation
  - Scrollable
